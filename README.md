## Technologies used

- React;
- Next.js;
- Redux;

## System Requirements

- Node.js 10.13 or later
- MacOS, Windows (including WSL), and Linux are supported


## Installing

- Fork and clone this repository
- Run `yarn install` in your terminal
- Run `yarn dev` starts this app in development mode
